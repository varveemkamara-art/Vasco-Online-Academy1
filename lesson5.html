Lesson 5: Keyboard and Mouse

Keyboard

The keyboard is an input device used to type letters, numbers, and symbols into a computer. It allows users to enter text, commands, and data.

Keyboards have different keys, including letter keys, number keys, function keys, and special keys such as Enter, Spacebar, and Backspace. These keys help users write documents, search the internet, and control programs.

The keyboard is commonly used for typing emails, writing school assignments, and entering passwords. It is one of the most important input devices for daily computer use.

Mouse

The mouse is an input device used to control the pointer on a computer screen. It helps users click, select, drag, and move items easily.

A mouse usually has buttons and a scroll wheel. The buttons are used to open files and programs, while the scroll wheel helps move up and down pages.

Using a mouse makes it easier to interact with icons, menus, and windows on the computer. It works together with the keyboard to perform tasks efficiently.

Summary

The keyboard and mouse are input devices that help users communicate with computers. They allow users to enter information, control programs, and complete tasks effectively.
